# src/assets — image notes

## storefront-placeholder.svg
A license-clean **illustration** of the Meat Center storefront (white metal
building, red gable roof, red "The Meat Center" script sign, central glass doors,
ICE machine on the right). Use it as the hero/exterior image while building.
**It is not a photograph — replace it before launch.**

## The real storefront photo (to drop in)
A genuine exterior photo exists on the business's Facebook page and was visually
verified during research:
- Post: https://www.facebook.com/photo.php?fbid=1025073890872200&set=pb.100064614372988.-2207520000&type=3
- Page: https://www.facebook.com/TheMeatCenter/ (Photos tab)
- Full resolution available: ~2048 x 1392 JPEG, posted Jan 29, 2016.
- Shows: white metal building, red roof, red script "The Meat Center" sign,
  parking lot, ICE machine.

### Why it isn't already here
It couldn't be saved automatically from this environment (image download was not
permitted via the tools available in this session). To add it:
1. Open the post URL above, right-click the photo → **Save image as…**
2. Save it as `src/assets/storefront.jpg`
3. Point the hero at `storefront.jpg` instead of `storefront-placeholder.svg`.

### Licensing reminder
This photo is owned by the business (or whoever posted it). It's fine for the
business's own site **once the owner confirms** you may use it. Best practice:
ask the owner for an original high-res exterior shot (and interior, the case, a
bundle/box deal, and a staff photo) — see `assets.md` in the project root.
